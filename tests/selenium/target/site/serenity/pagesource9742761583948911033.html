				"b8a4b3dbea5581e7d5b1704961ba8da683d2000f0789dffbca7ee968d0a39fc6"			);



			fetch(curl.getUrl(), {

				method: 'POST',

				headers: {'Content-Type': 'application/json'},

				body: JSON.stringify(request_data)

			})

				.then((response) => response.json())

				.then((response) => {

					if ('error' in response) {

						throw {error: response.error};

					}



					postMessageOk(response.success.title);



					if ('messages' in response.success) {

						postMessageDetails('success', response.success.messages);

					}



					this.disableNavigationWarning();



					const curl = new Curl('zabbix.php');



					curl.setArgument('action', 'dashboard.view');

					curl.setArgument('dashboardid', response.dashboardid);



					location.replace(curl.getUrl());

				})

				.catch((exception) => {

					clearMessages();



					let title;

					let messages = [];



					if (typeof exception === 'object' && 'error' in exception) {

						title = exception.error.title;

						messages = exception.error.messages;

					}

					else {

						title = this.dashboard.dashboardid === null || this.clone

							? "Failed to create dashboard"							: "Failed to update dashboard";

					}



					const message_box = makeMessageBox('bad', messages, title);



					addMessage(message_box);

				})

				.finally(() => {

					this.is_busy_saving = false;

					this.updateBusy();

				});

		},
